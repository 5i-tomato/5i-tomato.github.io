# 服务器核心

本服务器使用fabric作为服务器核心，并在此基础上使用mcdreforged守护服务器进程，实现插件挂载。此外，还使用了velocity作为反向代理核心。

# 游玩规范

Block Infinite Team (BIT) Minecraft 服务器公告

欢迎来到 Block Infinite Team (简称 BIT) 服务器！我们很高兴能与你分享这个独特的 Minecraft 世界，期待与你一同探索、建设和创新。以下是一些重要信息和规则，请所有玩家仔细阅读并遵守。

服务器信息：

  • 服务器名称: Block Infinite Team (BIT)
  • 服务器地址: mc.itomato.top
  • 主要玩法: 生存、建筑、红石

服务器规则：

  1. 严禁使用作弊手段破坏游戏平衡：包括但不限于矿物透视、boatfly，以及魔改的 mod、资源包等。
  2. 尊重他人：严禁人身攻击、辱骂、讥讽他人。
  3. 严禁讨论政治话题：在服务器有关的交流平台上，不允许讨论任何政治话题。
  4. 尊重他人作品：不得恶意损毁他人作品。

违规处理：

  • 任何违反以上规则的行为，将由群内成员讨论投票决定是否进行封禁处理。超过 1/5 的成员投票通过封禁即可生效（服务器管理员记作两票）。

活动与事件：

  • 本服务器会不定期举行各种活动，例如庆祝传统节日等，欢迎大家积极参与。

联系方式：

  • 服务器QQ群：785308242

希望大家在 Block Infinite Team (BIT) 服务器度过愉快的时光，与我们一同创造一个美好的 Minecraft 世界！

祝游戏愉快！

Block Infinite Team 管理团队

# 交流平台规范

Block Infinite Team (BIT) 交流平台规范公告

为了维护 Block Infinite Team (简称 BIT) 社区的和谐与公平，我们制定了一系列规范，请所有成员严格遵守。触犯以下规定者将被禁言或永久踢出，视情况严重程度进行处理。

严禁行为：

  1. 讨论或进行网络攻击、恶意的黑客行为、开挂
    • 严禁讨论或参与任何形式的网络攻击、恶意黑客行为以及使用作弊工具。
  2. 在服务器、活动中作弊、钻空子、破坏公平
    • 在服务器或任何相关活动中作弊、钻空子或任何破坏游戏公平性的行为都将受到严厉惩罚。

不建议行为：

这些行为视情况严重程度，可能会导致禁言或警告：

  1. 刻意炫耀或鄙视配置
    • 不要故意炫耀或贬低他人的设备配置，如帧数、内存、显卡等。
  2. 坑新
    • 严禁恶意误导新玩家或鄙视新玩家，这些行为可能会导致严重后果。
  3. 群名片
    • 群名片不得高仿他人、发布广告、含有误导性信息或控制字符。
  4. 破坏讨论环境
    • 禁止任何阴阳怪气、引战、刷屏或低质量发言等破坏讨论环境的行为。

适度行为：

这些行为需要略微注意，但过度也会受到处理：

  1. 梯子
    • 讨论梯子问题时不能发截图或分发相关资源，仅可适度讨论。
  2. 广告
    • 广告每五天只能发布一次，且必须与 Minecraft 相关。

违规处理补充说明：

  • 如果违反群规、故意作死或在管理劝阻以后依旧继续不适宜的话题，则可能被警告。如果严重违规或者多次被警告将会被永久踢出。
  • 群公告的群规只是一个衡量规则，就算某些行为群规中并没提到，但如果引人不适的话，依旧会被警告并处以相应惩罚。
  • 在你被警告时，大概率也会一起被禁言，禁言时间视恶劣程度不定。

联系方式：

  • 服务器QQ群：785308242

我们希望每个成员都能为维护 BIT 社区的良好氛围尽一份力，共同营造一个友好、公平和充满乐趣的交流平台。感谢大家的理解与合作！

Block Infinite Team 管理团队
